•	Сервер (для размещения бэкенда и веб-клиента) и SQL-сервер (для агента):
o	ОС: от Windows Server 2012 R2 с включённой ролью Hyper-V
o	.NET 5 Runtime (x64) и .NET 5 Hosting Bundle
o	Процессор: Intel Xeon / AMD EPYC
o	ОЗУ: от 1 ГБ
o	Доступ к MS SQL Server с полными правами
o	Диск: 200 МБ свободного места
•	Клиент:
o	Современный браузер: Chrome, Edge (поддержка HTML5, JavaScript)
o	Доступ к внутреннему IP-адресу сервера приложения
